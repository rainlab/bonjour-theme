Translating site messages

Messages can be found everywhere in this site. To translate these messages, sign in to your back-end administration area, then select from the navigation:

Settings > Translate > Translate Messages

Some messages are generated directly from their originating phrases — found inline with the markup — while others use special language keys. The language keys provided by this theme are located in the theme.yaml file.

Once you have translated some messages, move on to translating content by selecting an item from the Content menu.
